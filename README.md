# Water–Snake–Gun Game

Water–Snake–Gun is a small console game written in Python.  
It is based on the same idea as Rock–Paper–Scissors, where the player plays against the computer.

The game is simple, interactive, and runs completely in the terminal.

---

## Rules of the Game

- Snake beats Water  
- Water beats Gun  
- Gun beats Snake  
- If both choose the same option, the result is a draw

---

## How the Game Works

- The computer randomly selects one option: Water, Snake, or Gun  
- The player enters their choice through the keyboard  
- The program compares both choices  
- The result is printed on the screen

---

## Tools and Language

- Python 3  
- No external libraries used
